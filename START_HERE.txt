🔄 HOW IT WORKS
═══════════════════════════════════════════════════════════════════════════════

1. STUDENT SUBMITS APPLICATION
   └─ Fills: Reason, Date, Hours
   └─ All statuses = pending

2. ADVISOR REVIEWS
   ├─ If APPROVED → advisor_status = approved
   └─ If REJECTED → final_status = REJECTED ❌ (DONE)

3. HOD REVIEWS (only if advisor approved)
   ├─ If APPROVED → hod_status = approved
   └─ If REJECTED → final_status = REJECTED ❌ (DONE)

4. HOST REVIEWS (only if both approved)
   ├─ If APPROVED → final_status = APPROVED ✅ (DONE)
   └─ If REJECTED → final_status = REJECTED ❌ (DONE)

LOGIC:
  If (advisor=approved) AND (hod=approved) AND (host=approved)
    → final_status = APPROVED ✅
  Else Any rejection
    → final_status = REJECTED ❌
